# RepoLocator

### `Search through GitHub`

Just type in the name of the public repository and <br> the app searches on its own

### `View your public repositories`

There is an option to log in and to search through private repositories on your account.<br />

We do not save your password or any private data, we are no Facebook..

### `Made with React`

This project was made with `React` and `Redux`.<br />
